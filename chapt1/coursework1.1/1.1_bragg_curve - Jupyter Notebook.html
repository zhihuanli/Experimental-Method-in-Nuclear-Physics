      ## 作业 1.1 计算Bragg curve
      ​
      ​
      ### 日期
      - 限期一周
      ​
      ### 要求
      ​
      计算质子和$^{12}C$在水中的Bragg curve。分别调节质子和$^{12}C$的入射能量，使两者在水中的Bragg peak 都位于127mm处，Bragg curve的横轴单位取mm，纵轴单位取 MeV/mm。
      两种粒子的曲线画在一张图上。
      ​
      #### 按照两种方法计算，比较两种方法得到的结果
      ​
      1. 将材料切成厚度很小的薄片，此时可认为在薄片内dE/dx为常数
          - 能量为E的粒子穿过一个厚度为厚度$\Delta x_i$的材料后,能量损失$\Delta E_i$为，
      $$
      \Delta E_i=\frac{dE}{dx}(E) \cdot \Delta x_i
      $$
       
         - 粒子每经过一个切片,剩余能量为$E=E-\Delta E_i$, 穿过的厚度为：$x=\sum_i{\Delta x_i}$。由此得到Bragg curve上的一个点的坐标(x, $\frac{dE}{dx}(E)$)。以此类推，直至$E \to 0$。
          
          
      2. 假设粒子在材料中每次损失很小的固定能损$\Delta E$
          - 能量为E的粒子的粒子损失$\Delta E$的能量时，穿过的材料厚度$\Delta x$为，
      $$
      \Delta x=\frac{\Delta E}{\frac{dE}{dx}(E)}
      $$
          - 求出每一步的剩余能量$E$和穿过的材料厚度$x$，得到Bragg curve上的一个点的坐标(x, $\frac{dE}{dx}(E)$)。以此类推，直至能量$E'\to 0$

      作业 1.1 计算Bragg curve¶

      日期¶

      • 限期一周

      要求¶

      计算质子和12𝐶12C在水中的Bragg curve。分别调节质子和12𝐶12C的入射能量，使两者在水中的Bragg peak 都位于127mm处，Bragg curve的横轴单位取mm，纵轴单位取 MeV/mm。 两种粒子的曲线画在一张图上。

      按照两种方法计算，比较两种方法得到的结果¶

      1. 将材料切成厚度很小的薄片，此时可认为在薄片内dE/dx为常数

        • 能量为E的粒子穿过一个厚度为厚度Δ𝑥𝑖Δxi的材料后,能量损失Δ𝐸𝑖ΔEi为，
          Δ𝐸𝑖=𝑑𝐸𝑑𝑥(𝐸)⋅Δ𝑥𝑖ΔEi=dEdx(E)⋅Δxi
        • 粒子每经过一个切片,剩余能量为𝐸=𝐸−Δ𝐸𝑖E=E−ΔEi, 穿过的厚度为：𝑥=∑𝑖Δ𝑥𝑖x=∑iΔxi。由此得到Bragg curve上的一个点的坐标(x, 𝑑𝐸𝑑𝑥(𝐸)dEdx(E))。以此类推，直至𝐸→0E→0。
      1. 假设粒子在材料中每次损失很小的固定能损Δ𝐸ΔE
        • 能量为E的粒子的粒子损失Δ𝐸ΔE的能量时，穿过的材料厚度Δ𝑥Δx为，
          Δ𝑥=Δ𝐸𝑑𝐸𝑑𝑥(𝐸)Δx=ΔEdEdx(E)
        • 求出每一步的剩余能量𝐸E和穿过的材料厚度𝑥x，得到Bragg curve上的一个点的坐标(x, 𝑑𝐸𝑑𝑥(𝐸)dEdx(E))。以此类推，直至能量𝐸′→0E′→0

      TGraph* GetdEdxLise(string elossfile)
      {
          ifstream inf(elossfile);//读入LISE软件生成的12C入射到H2O的数据
          if(!inf.is_open()){     //文件不存时。。
              cout<<"Data File: "<<elossfile<< " does not exist!"<<endl;
              return 0;
          }
          auto *g=new TGraph;//定义新的二维散点图(TGraph) g
          string ss;
          double a, b, e, dedx;    
          getline(inf,ss);//读入第一行
          int i=0;
          while(inf >> a >> b >> e >> dedx >> a >> b >> a >> b >> a >> b >> a >> b) 
          {
              g->SetPoint(i++, e*12,dedx*1000);//添加数据点到图中，能量单位为MeV/u，对C需乘以12.
          }
          inf.close();
          return g;
      }
